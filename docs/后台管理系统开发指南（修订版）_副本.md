### 1.2 йЎ№зӣ®зӣ®ж Ү

йҖҡиҝҮжң¬йЎ№зӣ®пјҢдҪ е°ҶжҺҢжҸЎпјҡ

- еүҚеҗҺз«ҜеҲҶзҰ»жһ¶жһ„зҡ„ејҖеҸ‘жЁЎејҸ
- RBACпјҲеҹәдәҺи§’иүІзҡ„и®ҝй—®жҺ§еҲ¶пјүжқғйҷҗжЁЎеһӢ
- RESTful API и®ҫи®Ўи§„иҢғ
- JWT и®ӨиҜҒдёҺ Token еҲ·ж–°жңәеҲ¶
- Git еӣўйҳҹеҚҸдҪңе·ҘдҪңжөҒ
- Docker е®№еҷЁеҢ–йғЁзҪІ
- дјҒдёҡзә§д»Јз Ғи§„иҢғдёҺж–ҮжЎЈзј–еҶҷ

```java
        long startTime = System.currentTimeMillis();
        Object result = null;
        Throwable exception = null;
        try {
            result = point.proceed();
            return result;

}
```

### 7.1 дёәд»Җд№ҲйңҖиҰҒ Dockerпјҹ

**Docker и§ЈеҶізҡ„й—®йўҳпјҡ**

| й—®йўҳ | жІЎжңү Docker | жңү Docker |
|------|------------|----------|
| зҺҜеўғдёҚдёҖиҮҙ | "еңЁжҲ‘з”өи„‘дёҠиғҪи·‘" | дёҖж¬Ўжү“еҢ…пјҢеҲ°еӨ„иҝҗиЎҢ |

### 10.1 ејҖеҸ‘зҺҜеўғй—®йўҳ

#### Q1: Maven дҫқиө–дёӢиҪҪж…ўжҖҺд№ҲеҠһпјҹ

**A:** дҝ®ж”№ Maven е…ЁеұҖй…ҚзҪ®ж–Үд»¶пјҲ`~/.m2/settings.xml`пјүпјҢж·»еҠ йҳҝйҮҢдә‘й•ңеғҸ

> вҡ пёҸ жіЁж„Ҹпјҡ`<mirrors>` ж Үзӯҫ**еҸӘиғҪй…ҚзҪ®еңЁ `settings.xml`** дёӯпјҢдёҚиҰҒеҶҷеңЁ `pom.xml` йҮҢпјҒ

```xml
<!-- ~/.m2/settings.xml -->
<settings>
  <mirrors>
    <mirror>
      <id>aliyun</id>
      <n>Aliyun Maven</n>
      <url>https://maven.aliyun.com/repository/public</url>
      <mirrorOf>central</mirrorOf>
    </mirror>
```
