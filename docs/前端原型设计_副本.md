原型已搭建完成，覆盖了PRD中的全部6大模块，你可以直接在界面中点击左侧导航进行交互体验：

- **数据看板** — 4个KPI卡片 + 费用趋势折线图 + 工单状态饼图 + 故障排行 + 备件消耗排名
- **设备台账** — 设备列表（含状态标签）、搜索/筛选/导入导出/新增、二维码入口
- **工单管理** — 全部工单/待我处理/本月统计三个Tab，故障等级和工单状态标签联动
- **保养管理** — 保养工单与保养模板两个视图切换
- **备件管理** — 备件档案/入库/出库Tab切换，低库存预警横幅实时提示
- **系统管理** — 左侧子菜单（用户/角色/部门/菜单/日志），用户列表含角色标签
- **通知铃铛** — 右上角通知面板，区分未读/已读，按类型显示图标和颜色
